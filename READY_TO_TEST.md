## Troubleshooting

### If you still see errors:
1. Stop the server (Ctrl+C)
2. Restart: `npm run dev`
3. Wait for both frontend and backend to start
4. Hard refresh browser

### If formatting doesn't show:
1. Make sure you started a NEW conversation
2. Hard refresh browser (Ctrl+Shift+R)
3. Check browser console (F12) for errors

### If emojis don't render:
1. Update your browser to latest version
2. Try Chrome or Edge (best emoji support)
